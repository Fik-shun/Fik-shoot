# SPACE SHOOTER

Play the game [here](https://fik-shun.github.io/Fik-shoot/)

### Key Bindings

|   Key    |       Description        |
|----------|--------------------------|
|    W     |    Plane: move forward   |
|    S     |    Plane: move backward  |
|    D     |     Plane: move right    |
|    A     |     Plane: move left     |
|  Space   |       Shoot Missile      |
|  Shift   |           Boost          |
